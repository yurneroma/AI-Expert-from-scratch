## 为什么公开学习？

我相信最好的学习方式是：
1. **从零构建** - 理解每一行代码背后的原理
2. **公开分享** - 教是最好的学
3. **持续迭代** - 每天进步一点点

这个仓库记录我从神经网络基础到前沿研究的完整过程。
